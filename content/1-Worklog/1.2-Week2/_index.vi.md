### **Kết quả đạt được trong Tuần 2:**

* Nắm vững kiến thức nền tảng về AWS.  
* Hiểu cách hoạt động của AWS Support và AWS Budgets.  
* Thành thạo các khái niệm cơ bản của IAM, bao gồm:  
  - Users  
  - Groups  
  - Roles  
  - Policies  
* Cài đặt và cấu hình AWS CLI thành công với:  
  - Access Key  
  - Secret Key  
  - Default Region  
* Thực hiện được các lệnh CLI quan trọng như:  
  - Kiểm tra cấu hình  
  - Liệt kê IAM Users  
  - Liệt kê Regions  
* Quản lý tài nguyên AWS hiệu quả thông qua cả Console và CLI.
